# 🐱 猫咪咖啡馆 — Catfe

**像素风模拟经营 H5 游戏**  
玩家在街角经营一家猫咪咖啡馆，为顾客冲泡咖啡，同时照顾各怀绝技的猫咪们。

> **当前版本：v0.2.0** · 策划案 V2.0 · 基于 Phaser 3 + TypeScript + Vite 开发

---

### v0.2.0 核心功能

```
✨ 顾客 → 接单 → 🐱 吧台猫做咖啡 → 🐱 客座猫送餐 → ⭐ 品质评级 → 🪙 结算
                        ↑ 完整闭环，无需手动操作
```

- **🧑 顾客系统**：4 类顾客随机生成，耐心条管理，订单气泡交互
- **☕ 咖啡流水线**：队列制作 + 进度条 + 蒸汽动画 + 自动送餐
- **🐱 猫咪排班**：6 只猫 × 4 岗位（吧台/客座/收银/休息），技能自动生效
- **🚶 猫咪 AI**：猫→咖啡机取杯→端给顾客→返回岗位（三段式动画）
- **🤖 自动排班**：按技能等级一键分配所有猫咪
- **⭐ 品质评级**：四维评分（原料40%/时机30%/猫咪20%/设备10%），1-5星，每星+20%售价
- **🔥 完美状态**：连续5杯5星触发（10秒技能效果+30%）
- **🪙 经济系统**：金币结算（星级×招财加成）+ 声望惩罚
- **💾 存档**：LocalStorage 自动存/读/清
- **🎨 场景**：侧视像素风咖啡馆（吧台/桌椅/猫爬架/窗户/咖啡机）

## 游戏系统详解

### 🧑 顾客系统（`CustomerManager.ts`）

| 功能 | 说明 |
|------|------|
| 顾客类型 | 上班族（低耐心/高消费）、学生（高耐心/低消费）、退休人士（极高耐心）、游客（中耐心/高消费） |
| 生成规则 | 每 8 秒随机刷一位，自动分配到空座 |
| 入场动画 | 从屏幕上方走入座位（Back.easeOut 缓动） |
| 订单气泡 | 就座后显示饮品气泡（名称+价格），点击接单 |
| 耐心条 | 颜色绿→橙→红实时变化，归零则生气离店（声望 -5） |
| 自动接单 | 客座区有猫 → 气泡出现 1.5 秒后自动入队（无需点击） |
| 标签同步 | 顾客名字+耐心条每帧跟随精灵移动 |

### ☕ 咖啡制作流水线（`OrderManager.ts`）

```
① 顾客就座 → 弹出订单气泡（客座猫自动接单 / 手动点击）
② 吧台猫 → 走到咖啡机旁等候制作
③ 咖啡机制作（♨️ 蒸汽动画）→ 队列进度条跑
④ 制作完成 → 客座猫先走到咖啡机取杯 → 端到顾客桌前
⑤ 交付 → 金币结算 → 猫返回客座区 → 顾客满意离店
```

| 参数 | 值 |
|------|-----|
| 队列上限 | 初始 3 个 |
| 饮品时长 | 快速 5s / 中速 8s / 慢速 12s |
| 制作加速 | 吧台区「神速」技能最高 -15% 时长 |
| 送餐方式 | 自动触发，三段式动画（猫→咖啡机→顾客→返回） |

### 🐱 猫咪系统（`CatManager.ts`）

**6 只初始猫**：

| 名字 | 品种 | 性格 | 技能 | 最佳岗位 |
|------|------|------|------|---------|
| 团子 | 橘猫 | 慵懒 | 招财💰+15%、亲和力🪑-10% | 收银区 |
| 摩卡 | 黑猫 | 粘人 | 安抚🪑-5s、夜色守护💤+15% | 客座区 |
| 花花 | 三花 | 好奇 | 探索🪑-20%、人气💰+10% | 客座区 |
| 云朵 | 布偶 | 胆小 | 治愈🪑-15%、柔软💤翻倍 | 客座区 |
| 闪电 | 暹罗 | 调皮 | 神速☕+15% | 吧台区 |
| 小雪 | 白猫 | 粘人 | 净洁💤-30% | 休息区 |

#### 排班系统

| 岗位 | 图标 | 猫咪职责 |
|------|------|---------|
| ☕ 吧台区 | 咖啡机旁 | 制作咖啡（神速技能加速） |
| 🪑 客座区 | 桌椅旁 | 自动接单 + 端咖啡送餐（减耐心消耗） |
| 💰 收银区 | 吧台右侧 | 增加金币结算倍率（招财技能） |
| 💤 休息区 | 猫爬架 | 加速精力恢复 |

操作：
- 底部 🐱 **排班** → 点击槽位 → 选择空闲猫指派
- 再次点击已在岗猫 → **取消排班**（猫走回原位）
- 点击 **🤖 自动排班** → 按最高等级技能自动分配

#### 猫咪移动 AI

```
① 接单 → 吧台猫走向咖啡机等候制作
② 完成 → 客座猫走向咖啡机取杯（咖啡杯精灵出现）
③ 端咖啡走到该顾客桌前
④ 交付（杯子+蒸汽立即销毁）→ 金币结算
⑤ 猫走回客座区继续待命
```

- 名字标签每帧跟随精灵
- 取消排班自动走回初始位置
- 精力 < 20 时变半透明提示休息

#### 精力系统

- 工作区（吧台/客座/收银）每 5 秒 -2 精力
- 休息区每 5 秒 +3 精力（受技能加成）
- 精力不影响功能，仅半透明视觉提示

### 🪙 经济系统

- **金币公式**：基础价 ×（1 + 收银区技能倍率之和）+ 随机零头（≤30%）
- **招财示例**：团子 Lv1 +15%，花花 Lv1 +10%
- **声望惩罚**：顾客耐心归零 → 声望 -5

### ♨️ 咖啡机动画

- 制作中：咖啡机上方冒出 ♨️ 蒸汽（循环动画）
- 制作完成：蒸汽立即停止，触发送餐

### ⭐ 品质评级（`QualityRating.ts`）

每杯咖啡交付时根据四维评分评定星级：

| 维度 | 权重 | 影响因素 |
|------|------|---------|
| 🫘 原料品质 | 40% | 所有原料品级的加权平均（普通0.3/精品0.6/竞标1.0） |
| ⏱️ 时机加成 | 30% | 顾客耐心剩余比例越高分越高 |
| 🐱 猫咪加成 | 20% | 吧台区有猫 +0.5 星 |
| 🔧 设备等级 | 10% | 咖啡机等级提供品质加成（Lv1=0.25, Lv5=0.75） |

**星级效果**：

| 星级 | 总分范围 | 售价倍率 |
|------|---------|---------|
| ⭐ | <0.20 | ×1.2 |
| ⭐⭐ | <0.40 | ×1.4 |
| ⭐⭐⭐ | <0.60 | ×1.6 |
| ⭐⭐⭐⭐ | <0.80 | ×1.8 |
| ⭐⭐⭐⭐⭐ | ≥0.80 | ×2.0 |

**完美状态** 🔥：连续 5 杯获得 5 星 → 触发「完美状态」，10 秒内所有猫咪技能效果 +30%。

**视觉反馈**：交付时弹出星级文字（⭐弹跳动画 + ✨粒子散开），金币飘字同步显示星级。`StarDisplay.ts`

**经济影响**：最终金币 = 基础价 × 星级倍率 × (1 + 收银区技能倍率之和) + 随机零头
